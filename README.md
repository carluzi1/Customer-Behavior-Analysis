# Customer Shopping Behavior Analysis

## üìå Project Overview
This project analyzes customer shopping behavior using transactional data from **3,900 purchases** across multiple product categories. Using **Python (Pandas)** for data cleaning and analysis, **SQL (PostgreSQL)** for querying and aggregation, **Excel** for validation and exploratory checks, **Power BI** for interactive dashboards, and **Gamma AI** for presentation support, the project uncovers key insights into:

- Spending patterns  
- Customer segmentation  
- Product preferences  
- Subscription behavior  

The goal is to provide **data-driven insights** that support strategic business decisions and enhance customer understanding.
